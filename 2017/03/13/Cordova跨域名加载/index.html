Cordova跨域名加载

Mar 13, 2017点击

在使用cordova开发iOS应用的时候要注意的问题
我是在现有的项目上集入Cordova的，遇到的问题是本地的HTML文件内嵌入了外部超链接；
解决的办法：
完整做过iOS项目的应该都知道，在项目的info.plist文件中可以配置“应用交通安全设置” App Transport Security Settings，

在info.plist文件中添加一个Key 为App Transport Security Settings，类型为字典类型然后给它添加一个key为”Allow Arbitrary Loads in Web Content”，类型为布尔类型，值为YES，这个设置是允许任意的Web加载的，为了安全起见可以更暴力点，添加key 为”Allow Arbitrary Loads”,类型为布尔类型，值为YES。
以上问题花费了我三个多种才搞定，所以把经历写下来，给那些想了解Cordova的伙伴，希望你们少走点坑~~
